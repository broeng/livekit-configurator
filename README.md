Livekit Configurator
====================

Simple service for continually monitoring and configuring Livekit.

Currently focuses on Livekit SIP features, like trunks and dispatch rules.

Built for my own purposes, to avoid the **cli** tool in Kubernetes environments,
should not be considered production ready.

Running tests
-------------

Run:

```
nix flake check -L
```
